# **App Name**: DarkType

## Core Features:

- Real-time Typing Test: Presents a random set of words for the user to type against a timer.
- Accuracy Tracking: Monitors and calculates the accuracy of the user's typing, highlighting errors.
- Speed Calculation (WPM): Calculates and displays the user's typing speed in words per minute (WPM) in real time.
- User Profiles: Allow users to create profiles to save and track their typing progress over time.
- Leaderboard: Displays a leaderboard of top typists, encouraging competition.
- Adaptive Text Generation: Use an AI tool that dynamically adjusts the text difficulty based on the user's performance to provide a personalized challenge.

## Style Guidelines:

- Primary color: Dark charcoal (#333333) to maintain a dark theme aesthetic.
- Background color: Even darker grey (#1A1A1A) to provide a high contrast dark background.
- Accent color: Electric blue (#7DF9FF) to highlight interactive elements and results.
- Body and headline font: 'Space Grotesk' (sans-serif) for a techy feel; use 'Inter' (sans-serif) for longer text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Use minimalist, neon-style icons to complement the dark theme.
- A clean, full-screen layout that focuses user attention on the typing area. Reduce clutter to enhance focus.
- Subtle animations for word transitions and speed updates to provide a dynamic user experience.
